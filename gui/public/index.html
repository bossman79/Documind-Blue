Quota tracking

Counts are API calls (pages) per key. Session pages older than 4 hours without activity are dropped. Weekly pages older than 5 days without activity are dropped. A 429 snapshots current page counts into the pooled average, then resets that key. Unchecked keys do not update the pooled average. Remaining docs estimate uses average pages/doc from 429 hits.

Multiple API Keys — 429 failover, up to 40
On 429 rate limit, the job continues with the next key. Overrides single API Key when set. With local LLM-API-Key-Proxy, put your real Ollama keys here; use the single API Key field above for PROXY_API_KEY only.

Optional per-key HTTPS proxies (Ollama Cloud) are on the Per-key proxy tab.
